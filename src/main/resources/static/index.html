Aplicación de Suma y Resta

¿Cómo funciona?

Esta aplicación tiene dos operaciones básicas: suma y resta. Para realizar estas operaciones, solo necesitas ingresar dos números en la URL de la siguiente manera:

Suma: Para sumar dos números, ingresa la URL en el siguiente formato:

`http://localhost:8080/sum/{num1}/{num2}`

Por ejemplo, para sumar `2 + 2`, la URL sería:

`http://localhost:8080/sum/2/2`

Esto te devolverá el resultado: 4.

Resta: Para restar dos números, ingresa la URL en el siguiente formato:

`http://localhost:8080/resta/{num1}/{num2}`

Por ejemplo, para restar `5 - 3`, la URL sería:

`http://localhost:8080/resta/5/3`

Esto te devolverá el resultado: 2.

Recuerda que si intentas restar números donde el primero es menor o igual al segundo, recibirás un error. Para la suma, si los dos números son iguales, también se mostrará un mensaje de error.
